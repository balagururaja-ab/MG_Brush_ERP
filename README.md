# MG Brush ERP

MG Brush ERP is a desktop-based Enterprise Resource Planning (ERP) application developed for paint brush manufacturing businesses. It is built using Python, PostgreSQL, and Tkinter, following a layered architecture with Repository, Service, and UI layers.

## Features

- User Authentication
- Role-Based Access Control (RBAC)
- Company Management
- Supplier Management
- Customer Management
- Item & Category Management
- Purchase Management
- Sales Management
- Inventory & Stock Tracking
- Production Management
- Reports & Dashboard
- Audit Logs

## Technology Stack

- Python 3.14
- PostgreSQL
- Psycopg2
- Tkinter
- python-dotenv
- Git & GitHub
